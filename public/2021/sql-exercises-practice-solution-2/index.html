6、从不订购的客户

某网站包含两个表，Customers 表和 Orders 表。编写一个 SQL 查询，找出所有从不订购任何东西的客户。

Customers 表:

Id Name
1 Joe
2 Henry
3 Sam
4 Max

Orders 表：

Id CustomerId
1 3
2 1

应返回 | Customers | |———–| | Henry | | Max ||

select c.`Name` as `Customers` from Customers c left join Orders o on c.Id=o.CustomerId where o.Id is null;

1）本题扩展：查找不在表里的数据

查询在表A但不在表B的数据，使用SELECT … FROM 表A AS a LEFT JOIN 表B AS b ON a.列名=b.列名 WHERE b.列名 IS NULL;

7、部门工资最高的员工

Employee 表包含所有员工信息，每个员工有其对应的 Id, salary 和 department Id。

Id Name Salary DepartmentId
1 Joe 70000 1
2   Jim   90000   1            
3 Henry 80000 2
4 Sam 60000 2
5 Max 90000 1

Department 表包含公司所有部门的信息。

Id Name
1 IT
2 Sales

编写一个 SQL 查询，找出每个部门工资最高的员工。对于上述表，要求返回以下行（行的顺序无关紧要）。

Department Employee Salary
IT Max 90000
IT         Jim       90000  
Sales Henry 80000

下述语句的解题思路是：第一，把每个部门的最高薪资查出来作为一张临时表；第二，将员工表，部门表，部门最高薪资表做三个内连接（这里用左外连接也可，因为没有说有的员工的部门不存在的情况）

select d.`Name` as Department, e.`Name` as `Employee`, e.Salary from testEmployee e inner join (select DepartmentId, max(Salary) maxSalary from testEmployee group by DepartmentId) a on e.Salary=a.maxSalary inner join testDepartment d on e.DepartmentId=d.Id;

另外解法，把与部门最高薪资表的内连接换成IN函数

SELECT
   testDepartment.name AS 'Department',
   testEmployee.name AS 'Employee',
   Salary
FROM
   testEmployee
   JOIN
   testDepartment ON testEmployee.DepartmentId = testDepartment.Id
WHERE
   (testEmployee.DepartmentId , Salary) IN
   (   SELECT
          DepartmentId, MAX(Salary)
       FROM
          testEmployee
       GROUP BY DepartmentId
);

1）本题考点：GROUP BY

2）必须用GROUP BY生成一张临时表，因为可能同一个部门有两个员工都是最高薪资。采用下述语句是错的

select d.`Name` as `Department`, e.`Name` as `Employee`, e.Salary from Employee e, Department d where e.DepartmentId=d.Id group by e.DepartmentId having e.Salary=max(e.Salary);

下述错误

select * from testEmployee group by DepartmentId;

下述正确

select DepartmentId from testEmployee group by DepartmentId;

下述正确
